---
title: "Get Book by ID"
api: "GET /api/v1/books/findById/{id}"
description: "Retrieves a single book by its unique identifier (UUID)."
---

## Authentication

This endpoint requires a valid JWT token in the Authorization header.

```bash
Authorization: Bearer <your_jwt_token>
```

## Path Parameters

<ParamField path="id" type="string" format="uuid" required>
  The unique identifier (UUID) of the book to retrieve.
</ParamField>

## Response

<ResponseField name="id" type="string" format="uuid">
  Unique identifier of the book.
</ResponseField>

<ResponseField name="gender" type="string">
  The genre or category of the book.
</ResponseField>

<ResponseField name="author" type="string">
  The author's name.
</ResponseField>

<ResponseField name="image" type="string">
  URL or path to the book's cover image.
</ResponseField>

<ResponseField name="title" type="string">
  The main title of the book.
</ResponseField>

<ResponseField name="subtitle" type="string">
  The subtitle of the book.
</ResponseField>

<ResponseField name="publisher" type="string">
  The name of the publisher.
</ResponseField>

<ResponseField name="year" type="string">
  The publication year of the book.
</ResponseField>

<ResponseField name="pages" type="integer">
  The number of pages in the book.
</ResponseField>

<ResponseField name="isbn" type="string">
  The International Standard Book Number (ISBN).
</ResponseField>

## Status Codes

<ResponseField name="200" type="OK">
  Book retrieved successfully.
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Missing or invalid JWT token.
</ResponseField>

<ResponseField name="403" type="Forbidden">
  Valid token but insufficient permissions.
</ResponseField>

<ResponseField name="404" type="Not Found">
  Book with the specified ID does not exist.
</ResponseField>
